Comandos de Red

ping

Verifica la conectividad con otro equipo en la red.

ping google.com

Muestra el tiempo de respuesta del servidor.

ipconfig

Muestra la configuración de red en Windows.

ipconfig

Muestra IP, máscara y gateway.

ifconfig

Muestra interfaces de red en Linux.

ifconfig

Lista interfaces activas.

tracert

Muestra la ruta hasta un destino.

tracert google.com

Lista los saltos intermedios.

nslookup

Consulta información DNS.

nslookup google.com

Devuelve la IP del dominio.

netstat

Muestra conexiones activas.

netstat

Lista puertos y conexiones.
