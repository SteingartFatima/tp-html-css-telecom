Protocolos de Red

  1. HTTP Protocolo utilizado para transferir páginas web entre el navegador y el servidor. Funciona mediante solicitudes y respuestas.
  2. HTTPS Versión segura de HTTP que cifra la información utilizando SSL/TLS, protegiendo los datos transmitidos.
  3. FTP Protocolo empleado para la transferencia de archivos entre un cliente y un servidor dentro de una red.
  4. DNS Sistema que traduce nombres de dominio, como www.google.com, en direcciones IP numéricas.
  5. DHCP Protocolo que asigna automáticamente direcciones IP y otros parámetros de red a los dispositivos.
  6. TCP Protocolo orientado a conexión que garantiza la entrega correcta y ordenada de los datos.
  7. UDP Protocolo sin conexión que envía datos de forma rápida sin verificar la recepción.
  8. IP Protocolo encargado del direccionamiento y envío de paquetes entre dispositivos de una red.
