Red de Área Local (LAN)

¿Qué es una red LAN?

Una red LAN (Local Area Network) es un conjunto de dispositivos conectados entre sí dentro de un área geográfica limitada, como una casa, oficina o edificio.

Estas redes permiten compartir recursos como archivos, impresoras y conexión a internet entre los distintos equipos conectados.

Características principales

  • Alta velocidad de transmisión
  • Bajo costo de implementación
  • Área geográfica reducida
  • Administración centralizada
  • Permite compartir recursos
